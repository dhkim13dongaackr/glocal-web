## 2. 혁신방향(INNOVATION STRATEGY) 인터랙션

이 구간은 사용자가 스크롤하면서 각 혁신방향을 하나씩 집중해서 볼 수 있는 핵심 영역입니다.
저희 수정제안버전에 몇 가지 인터랙션을 적용해 봤는데, 공유드립니다.

### 2-1. 왼쪽 캠퍼스 이미지 전환

수정제안버전에서는 혁신방향이 바뀔 때마다 왼쪽 배경 이미지가 해당 캠퍼스 사진으로 크로스페이드 됩니다.

| 혁신방향 | 배경 이미지 |
|---------|-----------|
| 01 — Field Campus & R&D Innovation | 동서대 주례캠퍼스 |
| 02 — Settlement & Urban Boarding | 동아대 승학캠퍼스 |
| 03 — The Open Alliance University | 동아대 부민캠퍼스 |
| 지향점 — Connected Campus Ecosystem | 동서대 센텀캠퍼스 |

개발버전에서는 왼쪽 이미지가 고정되어 있는데, 카드 전환에 맞춰 배경도 함께 바뀌면 각 방향의 특성이 더 잘 전달될 것 같습니다.

### 2-2. 스크롤 스냅 & 중앙 고정

수정제안버전에서 적용한 방식입니다:

1. **섹션 진입 시 고정(Sticky)** — 혁신방향 영역에 도달하면 화면에 고정
2. **카드별 스냅** — 스크롤 시 카드가 항상 화면 중앙에 정렬, 카드 사이 어중간한 위치에서 멈추지 않음
3. **타임라인 진행** — 상단 도트가 현재 활성 카드를 표시하고, 진행선이 채워짐
4. **도트 클릭 이동** — 상단 도트를 클릭하면 해당 카드로 바로 이동

현재 개발버전에서는 스크롤 시 자연스럽게 지나가고 있어서, 스냅/고정을 적용하면 각 혁신방향에 대한 집중도가 높아질 것 같습니다.

### 2-3. 인디케이터 펄스 & 비활성 카드

- 활성 구간의 인디케이터(원형 포인트)에 은은한 펄스 효과를 넣으면, "지금 이 카드를 보고 있다"는 것을 직관적으로 알 수 있습니다
- 비활성(이전/다음) 카드를 반투명 처리하면 활성 카드와의 대비가 생겨 집중도가 올라갑니다

## 4. 성과 수치 (CORE ACHIEVEMENTS)

### 4-1. 숫자 + 단위 붙여쓰기

성과 수치가 하나의 덩어리로 읽히도록 '+'와 단위를 붙여 주시면 좋겠습니다.

| 현재 (개발버전) | 제안 |
|---------------|------|
| `15,000+ 명` | `15,000+명` |
| `2,000억+ 원` | `2,000억+원` |
| `7,000+ 명` | `7,000+명` |

### 4-2. 카운트 애니메이션

수정제안버전에서는 스크롤로 해당 구간에 진입하면 숫자가 0부터 올라가는 카운트업 애니메이션을 적용했습니다 (2초, ease-out cubic). 구간을 벗어났다가 다시 들어오면 애니메이션이 반복 재생됩니다.
개발버전에도 이 효과를 넣으면 성과 수치가 훨씬 임팩트 있게 전달될 것 같습니다.

### 4-3. 카드 배치

수정제안버전에서는 정방형(1:1) 영상 카드 3장을 가로로 나란히 배치하고, 각 카드 하단에 라벨+숫자를 오버레이하는 방식을 적용했습니다. 배경에 혁신성과 이미지를 soft-light 블렌드로 은은하게 깔아 분위기를 살렸습니다.

## 5. GLOCAL NOW

### 5-1. 레이아웃 — 타이틀 제한, 카드는 화면 끝까지

수정제안버전에서는 타이틀 영역은 content-w(1280px) 기준으로 좌측 정렬하되, 카드 영역은 오른쪽 화면 끝까지 확장되는 레이아웃을 적용했습니다.
개발버전에서도 이 방향을 검토해 주시면 좋겠습니다.

### 5-2. 스크롤 반응 속도

카드 이동 속도가 다소 느려서 스크롤 시 답답한 느낌이 있습니다. 감도와 속도를 조금 높여 주시면 더 쾌적할 것 같습니다.

### 5-3. 좌우 조작 버튼

우측 하단에 좌우 이동 버튼(`<` `>`)을 추가하면 스크롤 외에도 직접 탐색할 수 있어 편리합니다.
