## 1. 주요 변경사항 상세 명세

아래 항목들은 기존 사이트 대비 프리뷰 시안에서 변경 또는 추가된 동작입니다.
프로덕션 적용 시 참고 부탁드립니다.

### 1-1. 텍스트 변경

| 섹션 | 기존 | 변경 |
|------|------|------|
| 영문 섹션 타이틀 전체 | font-weight가 섹션마다 상이 (600~700) | **font-weight: 700(bold)로 통일**, font-size: 21px |
| INNOVATION 좌측 | "동아·동서 글로컬 연합대학" 24px→36px | **Figma 기준 36px→48px로 확대** |
| FIELD CAMPUS 설명 | "실무형 인재를 완성합니다" | **"실무형 인재를 키웁니다"** |
| CORE ACHIEVEMENTS 설명 | "글로컬 대학의 성과입니다" | **"글로컬 연합대학의 성과입니다"** |

### 1-2. 히어로 영상 — 스크롤 전체화면 전환

기존 사이트에서는 히어로 영상이 고정 크기이나, 프리뷰에서는 스크롤 시 전체화면으로 확장됩니다.

**동작 방식:**
- 초기: 16:9 비율 영상 카드 (좌우 패딩, `border-radius: 30px`)
- 스크롤 다운 시: `scale` 트랜스폼으로 뷰포트 전체를 채우도록 확장, `border-radius: 0`으로 전환
- 영상이 화면을 완전히 채울 때까지 sticky로 고정

**가로·세로 빈 영역 처리:**
- 영상 비율(16:9)과 뷰포트 비율이 다를 경우를 대비
- 뒤쪽에 **블러 처리된 동일 영상**을 깔아서 빈 공간을 자연스럽게 채움 (`filter: blur(40px) brightness(.6) saturate(1.4)`)
- 울트라와이드·세로 모니터 모두 대응

**슬라이드:**
- 5개 영상이 자동 전환 (5초 간격)
- 하단에 도트 인디케이터 + 일시정지/음소거 버튼

### 1-3. INNOVATION STRATEGY — 스크롤 기반 타임라인

기존: Framer Motion의 `whileInView`로 단순 페이드인
프리뷰: 스크롤 위치에 따라 단계별로 전환되는 타임라인

**동작 방식:**
- 좌측: 배경 이미지 (4장이 크로스페이드로 전환)
- 우측: 가로 타임라인 — 도트 4개 + 카드 4장이 현재 스텝 기준 중앙 정렬
- 스크롤 진행도(0~100%)를 5구간으로 나누어 각 스텝 활성화
- **프로그레스 라인**: 도트 중심 높이에 수평선, 진행도에 따라 `width: 0% → 50% → 100%`
- **도트 상태 3가지**:
  - 기본 (회색 `#c8c8d0`)
  - 지나간 스텝 (연보라 `#c4bff5`) — *기존 사이트에는 없던 표현*
  - 현재 활성 (보라 `#3E31D5` + 펄스 애니메이션)
- 도트 클릭으로 해당 스텝 직접 이동 가능

### 1-4. FIELD CAMPUS — 지도 + 카드 리스트

기존: 6개 카테고리 카드를 단순 나열
프리뷰: 부산 지도 위 핀 + 우측 카드 리스트 연동

**레이아웃:** 좌측 65% 지도 / 우측 35% 리스트

**3가지 상태:**

| 상태 | 지도 | 우측 패널 |
|------|------|-----------|
| 전체 (기본) | 21개+ 작은 보라 도트 모두 표시 | 안내 텍스트 ("카테고리를 선택해 보세요") |
| 카테고리 선택 | 해당 카테고리 핀만 표시, 나머지 숨김 | 해당 카테고리 캠퍼스 카드 리스트 (스크롤) |
| 캠퍼스 선택 | 선택 핀 강조(40px, 글로우) + 팝업 카드 | 해당 카드 하이라이트 (보라 테두리) |

**스크롤 연동:**
- 섹션 진입 시 "전체" 상태로 시작
- 스크롤에 따라 6개 카테고리가 순차 전환 (사용자가 탭을 직접 클릭하면 자동전환 비활성화)
- sticky 스크롤로 섹션 내에서 화면 고정

**스티키 스크롤 참고사항:**
- `position: sticky` 동작을 위해 부모 요소에 `overflow: hidden` 사용 불가
- 섹션 높이를 `250vh`로 설정하여 스크롤 여유 확보, 내부 콘텐츠는 `sticky top-0 h-screen`

### 1-5. CORE ACHIEVEMENTS — 배경 + 카운트 애니메이션

기존: 단색 배경 + 카운트업 1회 재생
프리뷰: 이미지 배경 블렌드 + 카운트업 반복 재생

**배경 처리:**
- 보라색 그라데이션(`rgba(62,49,213,0.92)`) 위에 혁신성과 이미지를 `background-blend-mode: soft-light`로 합성
- CSS `background` 속성 하나로 처리 (별도 오버레이 div 사용 시 sticky 동작이 깨질 수 있음)
- 이미지가 은은하게 비치는 정도 (약 8~10% 가시성)

**카운트 애니메이션:**
- 스크롤 진입 시 `0 → 목표값` 카운트업 (2초, ease-out 이징)
- 콤마 포맷 + 접미사: `15,000+`, `2,000억+`, `7,000+`
- **스크롤이 섹션 위로 되돌아가면 숫자를 0으로 리셋**, 다시 진입 시 재생
- 기존 사이트는 1회만 재생되어 놓치면 볼 수 없었음

**영상 카드:**
- 정방형(1:1) 비율, YouTube embed
- 하단 좌측 정렬 텍스트 (라벨 + 숫자 + 단위)
- 그라데이션 오버레이로 텍스트 가독성 확보

### 1-6. GLOCAL NOW — 가로 스크롤

기존: 일반 캐러셀 (좌우 버튼)
프리뷰: 세로 스크롤이 가로 이동을 구동

**동작 방식:**
- 섹션 높이 `250vh`, 내부 콘텐츠 `sticky`
- 세로 스크롤 진행도(0~100%)에 비례하여 카드 트랙을 `translateX`로 이동
- 카드가 뷰포트 밖으로 이어지며 스크롤에 따라 자연스럽게 흘러감
- 하단 우측에 `< >` 네비게이션 화살표도 병행 제공

### 1-7. FAQ — 독립 토글 + 로고 애니메이션

**아코디언 동작:**
- 기존: 하나를 열면 다른 항목이 자동으로 닫힘
- 변경: **각 항목이 독립적으로 열고 닫힘** — 여러 개를 동시에 펼쳐서 비교 가능

**대학 로고:**
- 동서대 로고: 우측 상단, `floatUp` 애니메이션 (2.4초, ±8px, 무한반복)
- 동아대 로고: 좌측 하단, `floatDown` 애니메이션
- `opacity: 6%`로 배경 장식 역할

### 1-8. 헤더 메뉴 호버

기존: 호버 시 텍스트 색상만 변경
프리뷰: **연보라 캡슐 배경**으로 감싸는 효과 추가

- 기본: `color: #171719` (검정), 배경 없음
- 호버/선택: `color: #3e31d5` (보라) + `background: #f0eeff` (연보라), `border-radius: 100px`
- 캡슐은 각 메뉴 컬럼 너비를 채우되, 높이는 텍스트에 맞게 컴팩트하게 유지

## 2. 추가 검토가 필요한 부분

### 모바일 대응
- 현재 시안은 데스크톱(1280px+) 기준으로 최적화되어 있음
- 모바일 뷰에서 Field Campus 지도, FAQ 2컬럼 등의 레이아웃 재배치가 필요
- 가로 스크롤 섹션(Glocal Now)의 터치 스와이프 대응 검토

### 히어로 영상
- 현재 3개 영상(president, hydrogen, semiconductor)만 포함 — contents, design 영상이 누락 상태
- 영상 파일 용량 최적화 검토 (현재 총 65MB → WebM 변환 시 절반 이하로 압축 가능)

### 이미지 최적화
- 일부 이미지가 원본 크기 그대로 사용 중
- WebP 변환 + 적정 해상도 리사이징으로 로딩 속도 개선 여지가 큼

### 콘텐츠
- Glocal Now 뉴스 카드의 실제 데이터 연동 (현재 정적 더미 데이터)
- FAQ 항목 내용의 실제 운영 데이터 반영
- Field Campus 캠퍼스별 상세 정보 보충

### 성능
- Tailwind CDN 런타임 빌드 → 프로덕션에서는 정적 CSS 빌드로 전환 권장
- YouTube iframe lazy load 적용 검토
- 폰트 로딩 최적화 (`font-display: swap` 등)
