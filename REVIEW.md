## 1. 이번 프리뷰에서 반영한 개선사항

### 헤더
- 메뉴 호버 시 연보라 캡슐 배경(`#f0eeff`) 효과 추가 — 선택 상태가 더 직관적으로 전달됨
- 드롭다운 메뉴 진입 시 배경 투명도 변화(`0.72 → 0.95`)로 가독성 확보

### INNOVATION STRATEGY
- 좌측 이미지 영역의 "동아·동서 글로컬 연합대학" 타이포 크기를 Figma 시안 기준(48px)으로 보정
- 영문 섹션 타이틀(`INNOVATION STRATEGY`) font-weight를 다른 섹션과 동일하게 통일 (700)
- 프로그레스 도트 타임라인에서 지나간 스텝을 연보라색으로 표시 — 진행 흐름이 더 명확해짐

### FIELD CAMPUS
- 부산 지도 기반 인터랙티브 맵으로 전면 교체 (기존: 단순 이미지 카드 나열)
- 6개 카테고리 필터 탭 + 21개 캠퍼스 핀 배치
- 핀 클릭 시 팝업 카드 + 우측 리스트 연동
- 스크롤 기반 카테고리 자동 전환

### CORE ACHIEVEMENTS
- 정방형(1:1) 영상 카드로 변경 — 시각적 임팩트 강화
- 텍스트 정렬을 하단 좌측으로 이동 (Figma 시안 반영)
- 숫자 콤마 포맷 + 접미사 적용 (`15,000+명`, `2,000억+원`, `7,000+명`)
- 배경에 혁신성과 이미지를 soft-light 블렌드로 적용
- 스크롤 재진입 시 카운트 애니메이션 반복 재생

### GLOCAL NOW
- Figma 시안 기반 카드 디자인 적용 (345px 카드, 240px 라운드 이미지)
- 스크롤 기반 가로 슬라이드 인터랙션 추가

### FAQ
- 2컬럼 레이아웃으로 재구성 (좌: 타이틀/설명, 우: FAQ 카드)
- 라운드 컨테이너(`border-radius: 40px`) + `#f5f7fa` 배경
- "Q." 보라색 접두사로 질문 강조
- 대학 로고 플로팅 애니메이션 (동아대, 동서대)
- 아코디언 독립 토글 (다른 항목 열어도 기존 항목 유지)

## 2. 추가 검토가 필요한 부분

### 모바일 대응
- 현재 시안은 데스크톱(1280px+) 기준으로 최적화되어 있음
- 모바일 뷰에서 Field Campus 지도, FAQ 2컬럼 등의 레이아웃 재배치가 필요
- 가로 스크롤 섹션(Glocal Now)의 터치 스와이프 대응 검토

### 히어로 영상
- 현재 3개 영상(president, hydrogen, semiconductor)만 포함 — contents, design 영상이 누락 상태
- 영상 파일 용량 최적화 검토 (현재 총 65MB → WebM 변환 시 절반 이하로 압축 가능)

### 이미지 최적화
- 일부 이미지가 원본 크기 그대로 사용 중 (동아대부민.jpg: 14MB, 동아대승학.jpg: 5.6MB)
- WebP 변환 + 적정 해상도 리사이징으로 로딩 속도 개선 여지가 큼

### 접근성
- 일부 인터랙티브 요소에 키보드 네비게이션 및 ARIA 속성 보완이 필요
- 영상 자동재생 시 접근성 가이드라인(WCAG) 준수 여부 확인

### 콘텐츠
- Glocal Now 뉴스 카드의 실제 데이터 연동 (현재 정적 더미 데이터)
- FAQ 항목 내용의 실제 운영 데이터 반영
- Field Campus 캠퍼스별 상세 정보(주소, 연락처 등) 보충

### 성능
- Tailwind CDN 런타임 빌드 → 프로덕션에서는 정적 CSS 빌드로 전환 권장
- YouTube iframe 3개 동시 로드 → Intersection Observer 기반 lazy load 적용 검토
- 폰트 로딩 최적화 (`font-display: swap` 등)

## 3. 기존 사이트 대비 구조적 차이점
